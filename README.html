Beverage Sales (Clustering & Classification)

Proyek ini bertujuan untuk menerapkan pendekatan unsupervised learning dan supervised learning dalam mengeksplorasi serta memprediksi pola dari data penjualan minuman. Dua pendekatan utama yang digunakan adalah Clustering dan Classification.

Clustering

Metode

  • Principal Component Analysis (PCA) untuk reduksi dimensi dan visualisasi data.
  • K-Means Clustering untuk mengelompokkan data berdasarkan kemiripan fitur.

Langkah-langkah

  1. Preprocessing Data: Normalisasi dan encoding fitur numerik dan kategorikal.
  2. Reduksi Dimensi: Menggunakan PCA untuk menyederhanakan visualisasi.
  3. Clustering: Menerapkan K-Means dan menentukan jumlah cluster optimal dengan Elbow Method dan Silhouette Score.
  4. Interpretasi: Menganalisis karakteristik tiap cluster setelah dilakukan inverse transform.

Classification

Model yang Digunakan

  • Random Forest Classifier
  • Logistic Regression
  • Gradient Boosting Classifier

Langkah-langkah

  1. Preprocessing:
    • Encoding fitur kategorikal (One-Hot atau Label Encoding)
    • Normalisasi fitur numerik
    • Pembagian data menjadi data latih dan data uji
  2. Training Model: Melatih ketiga model klasifikasi dengan data latih.
  3. Evaluasi:
    • Menggunakan metrik Accuracy, Precision, Recall, dan F1-Score.

Hasil

Ketiga model menunjukkan performa tinggi dengan hasil sebagai berikut:

  • Accuracy: 100%
  • Precision: 100%
  • Recall: 100%
  • F1-Score: 100%
